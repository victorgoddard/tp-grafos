Método Ford-Fulkerson – Algoritmo

1. para toda aresta e ∈ E(G) faça f(e) ← 0;       // Inicializar fluxo

2. Construir a rede residual G’(f)                // Construir rede residual inicial

3. enquanto existir algum caminho aumentante P em G’(f) efetuar

   a. Δ = min { uᵣ(e) | e ∈ P };                  // Determinar “gargalo” de P

   b. para cada aresta (v, w) ∈ P faça

      i. se (v, w) for aresta direta então
            f(v, w) ← f(v, w) + Δ                // Aumentar fluxo

      ii. senão
            f(w, v) ← f(w, v) − Δ                // Reduzir fluxo

   c. Atualizar a rede residual G’(f)             // Construir nova rede residual
